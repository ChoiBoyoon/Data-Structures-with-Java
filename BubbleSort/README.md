## Bubble Sort
> 버블 정렬. 데이터 집합을 순회하면서 집합 내의 이웃 요소들끼리의 교환을 통해 정렬을 수행.

* 느림! 요소의 수가 n개일 때, n(n+1)/2번의 연산 수행.
* 구현이 간단하기 때문에 버그를 만들 가능성이 적음.
