## Heap
> 힙 순서 속성(Heap Order Property)을 만족하는 완전 이진 트리

* 완전 이진 트리: 최고 깊이를 제외한 모든 깊이의 노드들이 완전히 채워져 있는 이진 트리
* 힙 순서 속성: 트리 내의 모든 노드가 부모 노드보다 커야 한다는 규칙. -> 힙에서 가장 작은 데이터를 갖는 노드는 부모 노드.
