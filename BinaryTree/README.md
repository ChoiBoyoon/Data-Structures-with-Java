## 이진 트리(Binary Tree)
> 모든 노드가 최대 2개의 자식을 가질 수 있는 트리.
> 특수한 자료구조. 자료를 담기 위해서가 아니라 아니라 컴파일러나 검색 등에 사용됨.

#### 사용 예
* **수식 이진 트리(Expression Binary Tree)**: 수식을 트리 형태로 표현하여 계산하게 해줌.
* **이진 탐색 트리(Binary Search Tree)**: 아주 빠른 데이터 검색을 가능하게 함.
